Create these files before running

```javascript
{
  "prefix":"<Desired prefix goes here>",
  "token":"<Paste bot token here>"
}
```

userData.json:
```javascript
{"users": {} }
```

pyconfig.py:

```python
userDataPath = "<Path to userdata.json>"

```
